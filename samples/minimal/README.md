# Minimal Sample

This is a minimal Revela project for testing and demonstration.

## Structure

```
minimal/
├── project.json    # Project configuration
├── site.json       # Site metadata
└── content/        # Add your photos here (create this folder)
```

## Usage

```bash
cd samples/minimal
mkdir content
# Add photos to content/

revela generate
```

## Configuration

- **Project:** minimal-sample
- **Theme:** default (built-in)
- **Output:** output/

Edit `project.json` and `site.json` to customize.
