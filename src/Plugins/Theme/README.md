# Spectara.Revela.Plugins.Theme

Theme management plugin for Revela — list, install, extract, and inspect themes.

## Commands

- `revela theme list` — List available and installed themes
- `revela theme files` — Show theme file structure
- `revela theme extract` — Extract theme files to project for customization
- `revela theme install` — Install a theme from NuGet
- `revela theme uninstall` — Uninstall a theme
- `revela config theme` — Configure theme settings
